# Adicione este favicon ao seu site

1. Renomeie o arquivo `pinheiro (1).png` para `favicon.png` ou `favicon.ico` para facilitar o uso.
2. Mova o arquivo para a raiz do projeto ou para a pasta `public/` se estiver usando Vite ou similar.
3. Adicione a seguinte linha dentro da tag <head> do seu `index.html`:

```html
<link rel="icon" type="image/png" href="favicon.png">
```

Se preferir usar a pasta `public/`, ajuste o caminho para `public/favicon.png`.
